Logos
-----

Except for a few files noted here, all of the logo files here are the original
assets from Yvonne K. Tang of Siliconian. The file `sxs_logo_notext.af` was the
result of importing the `.psd` (Adobe Photoshop) file into Affinity. The files
`sxs_logo_notext-replicated.{svg,pdf}` are the attempt of Leo Stein to replicate
the vector graphics in (Inkscape) SVG, and then exported to PDF.
